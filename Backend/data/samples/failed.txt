Job Summary:

Join our team as a Backend Developer, contributing to the design, development, and maintenance of scalable backend systems. You’ll use your Python and cloud expertise to build reliable microservices in a fully remote, collaborative environment while supporting high-impact, innovative products.


Key Responsibilities:

Build and maintain REST APIs and Python microservices (FastAPI/Flask).
Implement async workflows, background jobs, and task queues (Celery, Redis, etc.).
Design and optimize PostgreSQL schemas, indexes, and queries.
Integrate internal/external APIs (Slack, Google Sheets, third-party services).
Deploy and manage systems on AWS (EC2, S3, Route 53, CloudFront, Lambda).
Configure secure, high-performance Nginx reverse proxies.
Set up logging, monitoring, and alerting (CloudWatch, structured logs).


Required Qualifications:

Proven experience with Python REST APIs and microservices.
Strong async programming skills (Celery, Redis, etc.).
Deep PostgreSQL expertise.
Experience integrating APIs and automating workflows.
Solid AWS deployment/operations experience.
Practical Nginx configuration experience.
Clean, well-documented coding practices using Git.
Excellent written and verbal communication.


Preferred Qualifications:

Experience with LLM APIs (Gemini, Claude, GPT).
Prompt engineering and agentic workflow design skills.
Passion for automation, AI, and advanced backend solutions.
